What is jQuery?


jQuery: Open Source JavaScript Library • Simplifies interactions between HTML/CSS document and JavaScript. • Facilitates HTML document traversing and manipulation. • Handles browser event handling. • Facilitates DOM animations. • Facilitates Ajax interactions. • Aids in cross-browser JavaScript development.


jQuery Philosophy and Concepts

"Write less, do more" philosophy: - Find elements via CSS selectors and use jQuery methods. - Chain multiple jQuery methods on a set of elements. - Use jQuery wrapper and implicit iteration.


Using jQuery (JS) library on HTML page

There are two ways to start using jQuery on your web site.

  1. you can download the jQuery file on your local machine and include it in the HTML page.
  2. - you can add the jQuery library into your HTML page directly from a Content Delivery Network (CDN).

1. By download the jquery file locally :-


  • Visit https://jquery.com/download/ for latest version.
  • downloaded jQuery-3.6.0.min.js in website directory.
  • Link jQuery in HTML page using script tag and src attribute.

2.By Link jquery from google CDN

We can link the jQuery library in our HTML page directly from a Content Delivery Network. There are different CDNs that provide the latest version of the jQuery library.


Advantages:


  • Ease of use
  • Allows developers to create plug-ins on top of the JavaScript library.
  • Large development community.
  • Comprehensive documentation.
  • Easier to use than standard JavaScript.
  • Enables development of Ajax templates for a sleeker interface.
  • Lightweight and powerful chaining capabilities.

Disadvantages:


  • JQuery's extensive library may limit functionality, making raw JavaScript use inevitable.
  • The JQuery javascript file is required for running JQuery commands, despite its small size (25-100KB).
  • This file strains the client computer and potentially the web server if hosted on a personal web server.

How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , JQuery Animation?


To apply CSS properties using jQuery, use the css() method. It’s easy to apply any CSS property using jQuery method css( PropertyName, PropertyValue ).


Here is the syntax for the method −
$().css(propertyname, value);
$().css(properties);
example :
$(document).ready(function(){
$(".in").css("border","2px solid red")
})

addClass() method:

To add one or more class names to the chosen element, use the addclass() function. This technique is not intended to erase the current class attributes; rather, it is only intended to add one or more class names to the class attributes.

.addClass(className, [duration], [easing], [callback])

removeClass() method:


The set of matching elements is cleared of all or the designated class using the removeClass method.

Syntax:


removeClass(className, [duration], [easing], [callback])

animate() method

The jQuery animate() method provides you a way to create custom animations.


Syntax:


(selector).animate({styles}, para1, para2, para3);
example :
<script>
$(document).ready(function(){
$("button").click(function(){
$("div").animate({left: '450px'});
});
});
</script>
